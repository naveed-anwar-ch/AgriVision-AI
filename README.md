# 🌿 AgriVision AI - Plant Disease Detection System

## 📖 Overview

AgriVision AI is an intelligent plant disease detection system that leverages Deep Learning and Transfer Learning techniques to accurately identify plant diseases from leaf images. The system provides real-time disease prediction, webcam-based detection, AI-powered agricultural assistance, and prediction history tracking through a user-friendly Streamlit interface.

---

## 🎯 Project Objectives

* Detect plant diseases from leaf images.
* Assist farmers in identifying crop health issues.
* Provide real-time disease predictions.
* Enable live webcam-based disease detection.
* Offer AI-powered agricultural guidance through Agri Bot.
* Maintain a history of previous diagnoses.

---

## 🚀 Features

### 🔍 Plant Disease Prediction

* Upload a plant leaf image.
* AI model predicts the disease category.
* Displays confidence score.

### 📷 Live Webcam Detection

* Capture leaf images directly from webcam.
* Instant disease analysis.

### 🤖 Agri Bot

* AI-powered agriculture assistant.
* Answers farming and crop-related questions.

### 📜 Prediction History

* Stores previous predictions.
* Allows users to review past diagnoses.

### 👤 User Authentication

* Secure login and registration system.

## ⚙️ Installation

### Clone Repository

git clone https://github.com/naveed-anwar-ch/AgriVision-AI.git

### Create Virtual Environment

python -m venv plant_env

### Activate Environment

plant_env\Scripts\activate

### Install Dependencies

pip install -r requirements.txt

### Run Application

streamlit run app.py
